### 1.3 TC-FIX-FONT-02 失败原因分析

**失败信息**:
```
Component 'streamlit-echarts.streamlit_echarts' must be declared in pyproject.toml with asset_dir to use file-backed js.
```

**根本原因**:
- test_fixes.py 通过 `importlib.util.spec_from_file_location` 导入 app.py
- app.py 第69行导入 `from streamlit_echarts import st_pyecharts`
- streamlit-echarts 组件在非 Streamlit 运行环境下触发配置错误

**独立验证结果**:
通过直接测试字体查找逻辑（不导入 app.py），验证结果：
- macOS 系统成功找到 2 个中文字体
- 首选字体: `/System/Library/Fonts/STHeiti Light.ttc`
- 字体查找功能**完全正常**

**结论**: 字体功能本身无问题，测试失败是由于测试方法不当（在非 Streamlit 环境导入 Streamlit 应用）
